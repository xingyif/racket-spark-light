4 Creating A Datashell

A Datashell is an immutable structure that stores user inputted lists of data, on which RSL Operations are performed.

You will able to create a Datashell with the following functions:

syntax

(mk-datashell l)

 
  l : list?
Creates a Datashell from the given list.

Example:

(define a (mk-datashell '(5 2)))

syntax

(mk-datashell path)

 
  path : String?
Creates a Datashell from the given csv file path.

Example:

(define csv-a (mk-datashell "../path-to-file.csv"))

All Transformation Functions (TFuncs) take one input, and have one output
